## Measurements & Specifications

### Container
- Max width: 1200px
- Padding: 0 120px (sides)
- Centered alignment

### Hero Section
- Height: 560px
- Background: Gradient or hero image
- Search bar: 600px wide
- Filter buttons: 140px each

### Property Cards
- Width: 380px
- Height: 420px
- Image height: 240px
- Padding: 20px
- Border radius: 12px
- Shadow: 0 2px 8px rgba(0,0,0,0.08)

### Grid System
- Columns: 12
- Gutter: 24px
- Margin: 120px (sides)

### Typography
- Hero title: 48px, bold
- Hero subtitle: 20px, regular
- Section titles: 32px, semibold
- Card price: 24px, bold
- Card address: 14px, regular
- Body text: 16px, regular

### Spacing
- Between sections: 64px
- Between cards: 24px
- Internal card padding: 20px
- Button padding: 16px 32px

### Colors (Reference)
- Primary: #6366F1
- Text primary: #1A1A1A
- Text secondary: #6B7280
- Border: #E5E7EB
- Background: #FFFFFF
- Card background: #FFFFFF

## Responsive Notes
- At 1024px: Reduce to 2 property cards per row
- At 768px: Switch to tablet layout
- At 640px: Switch to mobile layout
- Search filters stack vertically on mobile
